Operator

Jenis operator pada javascript:

  • Aritmatika. KuKaBaTaKu () * / + -
  • Penugasan
  • Perbandingan
  • Logika
  • String
  • Kondisional

Operator binary: Aritmatika, penugasan, perbandingan, logika, string. Membutuhkan 2 operand: operand-1 operator operand-2

Operator ternary: Kondisional. Membutuhkan 3 operand

Operator unary: typeof. Membutuhkan 1 operand: operator operand atau operand operator
